contribute

Your help is very appreciated!

We are always improving LipeRMI and we need help from others enthusiastics like us.

You can contribute in many ways, like:

  • using LipeRMI in your application (please, let we know!);
  • writing documentations;
  • sending suggestions;
  • reporting bugs / sending patches;
  • donating some money.

Also, feel free to improve LipeRMI on your way. Download source code here.
